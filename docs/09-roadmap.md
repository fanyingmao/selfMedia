## 9.3 M1 · MVP

**目标**：1 篇长文 → 1 条 60 秒竖版口播短视频 → 上传到 1 个平台（YouTube Shorts，OAuth 最稳）。

**包含**：
- 用户：邮箱注册 / 登录、单组织。
- 素材：URL 抓取 + 文件上传（PDF / DOCX / MD）。
- 流水线：understand → ideate → script → storyboard → image（仅文生图）→ tts → render。
- 模型：LLM 用 GPT-4o-mini / Claude Haiku；图用 Flux / SDXL；TTS 用 Azure 或 ElevenLabs。
- 编辑器：仅"标题 + 字幕 + 重生成单镜头图片"三件套。
- 发布：YouTube Shorts（OAuth）。
- 计费：免费额度 + 简单按量记录（不收费）。
- 可观测：基础日志 + 简单指标。

**不包含（明确推后）**：文生视频、多平台、风格 LoRA、矩阵、数据回流、协作。

**完成判据**：
- 端到端 P95 ≤ 8 分钟。
- 50 条种子内容人工评审"可用率" ≥ 70%。
- 单条成本 ≤ 1 美元。

## 9.4 M2 · V1（公开测试版）

**目标**：覆盖创作者最高频的 3 个国内平台 + 真正"工作台"体验。

**新增**：
- 平台：抖音 / 小红书 / 视频号（先做发布，难点平台用 Playwright 兜底）。
- 编辑器：完整时间线（拖拽 / 替换素材 / 替换配音 / 调时长 / 封面编辑）。
- 风格预设：内置 10 套（口播 / 综艺 / 电影感 / 卡通 / 国风 / 信息流 / 二次元 / 简约 / Vlog / 教程）。
- 文生视频：接入可灵 / 即梦 / Runway 任意 1 家，仅用于"高光镜头"。
- 审核：文本敏感词 + 图像 NSFW，平台合规标识。
- 计费：套餐 + 按量；接入 Stripe / 支付宝 / 微信。
- 通知：站内 + 邮件 + Webhook。

**完成判据**：
- 公开 Beta 1000 用户；周留存 ≥ 30%。
- 单条 P95 ≤ 5 分钟、成本 ≤ 0.6 美元。
- 关键 SLO 达标（见部署文档）。

## 9.5 M3 · V2（账号矩阵 + 数据回流）

**目标**：让"1 人管 50 号"成立。

**新增**：
- 账号矩阵：批量绑定、分组、定时分发、风控（IP / 指纹 / 频率）。
- 数据回流：抖音 / 小红书 / 视频号 / B 站 / YouTube 数据 API + 爬虫兜底，每日聚合。
- 选题反训：基于回流数据训练"爆款选题打分模型"（先离线评估 → 再上线 AB）。
- 私有风格库：用户上传 30 张图 / 1 段视频，训练专属 LoRA / 风格 embedding。
- 多语言：英 / 日 / 泰，端到端打通（含 TTS 与字幕）。
- 团队基础：组织内 2~5 人协作 + 角色权限。

**完成判据**：
- 单租户支持 100+ 账号矩阵稳定调度。
- 选题打分模型相比 baseline CTR 提升 ≥ 15%（A/B）。

---

## 9.6 M4 · V3（团队 + 企业版）

**目标**：进入 MCN 与企业市场。

**新增**：
- 协作：CRDT 编辑、评论、@、审批工作流。
- 品牌词库 / 黑白名单：企业用。
- 私有部署版（Helm + 离线许可证）。
- 自托管模型：vLLM 跑开源 LLM，ComfyUI 跑 SDXL/Flux，Whisper 自托管。
- 模型微调：基于企业历史素材微调"语气 / 文风"小模型（LoRA 级别）。
- SOC 2 / ISO 27001 / 等保合规准备。

**完成判据**：
- 至少 5 个付费 MCN / 企业客户上线。
- 私有部署交付时长 ≤ 1 周。

---

## 9.7 M5 · Platform（开放生态）

**目标**：从"工具"演化为"平台"。

**新增**：
- 开放 API + 计量计费 + 文档站。
- 插件 / 应用市场（自定义节点接入流水线，例如"行业知识库"、"特殊审核器"）。
- 模板市场（创作者发布风格预设、Prompt 模板，付费 / 免费）。
- 数据接入伙伴（行业素材库、热点接口）。

## 9.8 风险与对策

| 风险 | 对策 |
| --- | --- |
| 各平台 API 政策变化 / 风控 | 多通道（官方 + 兜底）+ 主动监控 ToS 更新 |
| 模型成本上涨 / 接口变更 | model-adapter 抽象 + 多供应商 + 缓存 / 批量 |
| 内容合规风险 | 多层审核 + AIGC 标识 + 人工兜底 + 法务回顾 |
| GPU 资源紧张 | Spot + 多区域 + 队列优先级 + 关键时段提前扩容 |
| 同质化竞争 | 数据回流闭环 + 私有风格 + 行业垂直版 |
| 平台账号封禁 | 矩阵分散 + 频率与指纹策略 + 用户教育 |

## 9.9 评估指标（北极星 + 关键指标）

- **北极星**：每周成功发布短视频数（W-Publish）。
- 输入侧：周活创作者数、新建草稿数。
- 转化侧：草稿 → 发布 转化率、发布成功率。
- 价值侧：用户内容总播放量、节省工时（自评 + 抽样）。
- 商业侧：付费转化率、ARPU、毛利率（扣模型成本后）。
