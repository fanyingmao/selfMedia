# 10 · 安全、合规与成本控制

## 10.1 安全

### 身份与访问
- 用户：邮箱 / 手机号 + OAuth（Google / GitHub / Apple / 微信 / 飞书）。
- 多因素认证（TOTP / Passkey）：付费版强制开启。
- 权限：组织 → 角色（owner/admin/editor/viewer）→ 项目级 ACL。
- 服务间：mTLS + ServiceAccount JWT。
- API Key：可生成 / 撤销 / 限作用域 / 限 IP。

### 数据安全
- 静态加密：DB、对象存储默认 KMS 加密。
- 传输加密：HTTPS 全链路；内部 mTLS 可选。
- 敏感字段：平台账号 token、TTS 声纹样本，使用列级加密（Vault Transit）。
- 审计：管理类操作、平台发布、计费变更全部落审计表。

### 应用安全
- OWASP Top 10 评审；依赖漏洞自动扫描（Trivy / Snyk）。
- CSP、SameSite、CSRF Token、XSS Filter。
- 渗透测试：上线前 + 每年 1 次第三方。

## 10.2 合规

### 内容合规
- 文本：自建敏感词库（涉政 / 涉黄 / 涉暴 / 医疗金融）+ LLM 二次判定 + 国内云内容安全 API。
- 图片 / 视频：NSFW、明星 / 政要人脸、Logo 识别、商标图案识别。
- AIGC 标识：
  - 视频右下角自动烧录 "AI 生成" 标识。
  - 平台元数据按各平台规范打标（如视频号 / 小红书的 AIGC 申报）。
- 版权：
  - 引用原文须支持"原文链接 + 作者署名"。
  - BGM 仅用授权曲库 / 自生成；禁用平台音乐版权之外的曲目。
  - 用户上传素材需勾选"我拥有版权"声明。

### 数据合规
- 中国大陆：合规接入"数据出境安全评估"流程；个人信息单独同意 + 最小化收集。
- 欧盟 GDPR：数据导出 / 删除 / 同意管理；DPA 模板。
- 加州 CCPA：Do-Not-Sell；消费者权利接口。
- 未成年人：禁止 < 14 岁注册（按地区区分）。

### 平台合规
- 维护各平台 ToS 监控（专人 + 工单）。
- 自动化发布频率与人类节奏对齐；提供"非自动化模式"。
- 账号绑定明确告知用户风险；平台官方 API 优先于浏览器自动化。

## 10.3 成本控制

成本由 4 大块构成：**LLM Token、图像生成、视频生成、TTS / 音频**；其次是 GPU 渲染、出口带宽、对象存储。

### 按租户预算
- 套餐内额度 + 超出按量计费。
- 实时 `usage_events` 流入 → 当日预算监控 → 超限自动降级到 cheap_pool。
- 用户可设置"硬上限"，超过则暂停而不是继续扣。

### 模型路由优化
- 任务分级：摘要 / 实体抽取 → cheap_pool；选题 / 文案 / 分镜 → top_pool。
- 长上下文：> 30k token 才走 long-context 模型，避免误用昂贵模型。
- 提示压缩：摘要 + 关键信息抽取后再传给下游模型。
- Embedding 复用：素材级缓存，避免重复嵌入。

### 媒资成本
- 文生视频按需启用，默认仅"高光镜头"使用；其余镜头用文生图 + Ken Burns。
- 内容寻址缓存（CAS）：相同 prompt + 相同 seed + 相同 model 命中缓存直接复用。
- 渲染节点 Spot 优先；峰值才上按需。
- 对象存储分级：成片标准存储；中间产物 30 天后归档（IA / 冷归档）。

### 计量准确性
- 每次模型调用 Adapter 强制返回 `{tokens_in, tokens_out, images, seconds}`，落 `usage_events`。
- 内部成本字典 `unit_cost` 与对外计费字典 `bill_cost` 分离，便于调价和促销。
- 月末与厂商账单对账，差异 > 1% 自动告警。

### 运维成本
- 自动扩缩（HPA / KEDA）；闲时回收 GPU。
- 多模型多区域：避开 OpenAI / Anthropic 速率峰值，海外 + 国内多通道。
- 高 IO 任务（视频拼接）尽量与对象存储同区域，避免出口流量费。

## 10.4 风险清单（合并）

| 类别 | 风险 | 缓解 |
| --- | --- | --- |
| 安全 | 模型 API Key 泄漏 | Vault + 短期 Token + 出现立即轮换 |
| 安全 | 用户上传恶意文件 | 病毒扫描 + 沙箱解析 + 限格式 / 限大小 |
| 合规 | 误生成违规内容 | 多层审核 + 人工兜底 + 一键召回 |
| 合规 | 平台账号被风控 | 频率 / 指纹 / 真人节奏 + 用户告知 |
| 成本 | 单租户跑量 | 硬上限 + 实时预警 + 自动降级 |
| 成本 | 模型涨价 | 多供应商 + 月度议价 + 本地化候选 |
| 业务 | 数据回流接口失效 | 官方 + 爬虫双通道 + 监控 |
| 业务 | AI 质量波动 | 评测集 + AB + 灰度 + 一键回滚 prompt 版本 |
