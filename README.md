# SelfMedia · AI 自媒体内容生产工具

> 一站式 AI 自媒体内容生产平台：把一篇长文（公众号文章 / 论文 / 新闻 / 博客 / 报告）一键转化为「短视频脚本 + 配套图片 / 短视频素材」，并支持多平台一键分发。

---

## 1. 项目简介

SelfMedia 是面向个人创作者、MCN 机构和企业新媒体团队的 **AI 内容工厂**。它解决一个核心痛点：

> "我有海量优质长文素材，但没时间把它们改写成抖音 / 小红书 / 视频号能用的短视频内容。"

平台基于 LLM + 文生图 + 文生视频模型，把"长文 → 选题 → 短视频文案 → 分镜脚本 → 图片素材 → 短视频成片 → 一键发布"做成一条全自动可干预的流水线。

## 2. 核心能力一览

| 能力模块 | 关键功能 |
| --- | --- |
| 素材采集 | URL 抓取、PDF / Word / Markdown 导入、RSS 订阅、剪贴板粘贴、知识库沉淀 |
| 长文理解 | 自动摘要、关键观点抽取、人物 / 时间 / 地点实体识别、情感与受众分析 |
| 选题策划 | 基于热搜 / 平台调性的"爆款选题"重写，多版本 A/B 标题生成 |
| 短视频文案 | 钩子开头、口播逐字稿、字幕分句、Hashtag、平台风格适配（抖音 / 小红书 / B 站 / 视频号 / YouTube Shorts） |
| 分镜脚本 | 镜头切分、画面描述、镜头时长、BGM 建议、转场建议 |
| 图片生成 | 文生图（封面图、分镜图、信息图）、图生图、风格统一控制（LoRA / 风格预设） |
| 短视频生成 | 文生视频片段、图生视频、字幕烧录、TTS 配音、BGM 合成、自动剪辑成片 |
| 多平台发布 | 抖音 / 小红书 / 视频号 / B 站 / YouTube / X 等平台账号矩阵管理与定时发布 |
| 数据回流 | 播放、点赞、评论、转化数据回流，反向训练选题模型 |

## 3. 目标用户

- **个人创作者**：1 人想做出 5 人的产能。
- **MCN 机构**：批量孵化账号，需要把同一选题"千号千面"。
- **企业新媒体 / 品牌运营**：把白皮书、产品文档变成短视频投流素材。
- **知识付费讲师**：把课程长文切成短视频引流钩子。

## 5. 快速预览：一条内容的生命周期

```
长文 URL / 文件
   │
   ▼
 [采集服务] ──► 原文清洗、分段、向量化入库
   │
   ▼
 [理解服务 LLM] ──► 摘要 / 关键观点 / 实体 / 受众画像
   │
   ▼
 [选题与文案 LLM] ──► 多版本短视频文案 + 标题 + Hashtag
   │
   ▼
 [分镜服务 LLM] ──► 逐镜头 prompt（画面 / 时长 / 字幕 / 旁白）
   │
   ├──► [文生图服务] ──► 分镜图 / 封面图
   ├──► [文生视频服务] ──► 镜头视频片段
   ├──► [TTS 服务] ──► 旁白音频
   └──► [BGM / SFX 服务] ──► 背景音乐
   │
   ▼
 [合成服务 FFmpeg] ──► MP4 成片 + 字幕 + 封面
   │
   ▼
 [人工微调台] ──► 创作者在 Web 端二次编辑
   │
   ▼
 [发布服务] ──► 抖音 / 小红书 / 视频号 / B 站 / YouTube
   │
   ▼
 [数据回流] ──► 播放 / 互动数据 → 训练选题模型
```
